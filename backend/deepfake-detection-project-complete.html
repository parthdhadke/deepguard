📋 Project Overview

A complete end-to-end deepfake detection system built for academic research and deployment. This project implements MesoNet, a lightweight convolutional neural network specifically designed for detecting AI-generated or manipulated face images.

Project Type: Image and Video Processing Class Project
Author: Parth Dhadke
Institution: Mumbai University, B.Tech Computer Science & Data Science (Batch 2027)
Training Time: 2-3 hours on NVIDIA GPU
Model Size: ~1MB (28,754 parameters)

Key Capabilities

  • Image Detection: Analyzes single face photos with confidence scores
  • Video Detection: Processes video frame-by-frame with temporal analysis
  • Face Extraction: Automatic face detection and cropping using Haar Cascade
  • Report Generation: Provides detailed text explanations for each prediction
  • REST API: FastAPI backend with Swagger UI for easy testing
  • GPU Acceleration: ~10ms inference per image on NVIDIA GPU

🏗️ System Architecture

MesoNet Model Architecture

MesoNet is a mesoscopic CNN designed specifically for deepfake detection, focusing on medium-scale facial features that reveal AI manipulation artifacts.

Layer Input Size Output Size Parameters
Input 150×150×3 - -
Conv Block 1 (3→8) 150×150×3 75×75×8 224
Conv Block 2 (8→8) 75×75×8 37×37×8 1,616
Conv Block 3 (8→16) 37×37×8 18×18×16 3,216
Conv Block 4 (16→16) 18×18×16 9×9×16 6,416
Flatten 9×9×16 1,296 -
Dense + Dropout 1,296 16 20,752
Output 16 2 34

Total Parameters: 28,754

Data Flow Pipeline

  1. Input Reception: FastAPI receives image/video upload via POST /predict
  2. Face Detection: Haar Cascade detects and crops face with 20% padding
  3. Preprocessing: BGR→RGB conversion, resize to 150×150, normalize with ImageNet stats
  4. Model Inference: MesoNet forward pass on GPU (~10ms)
  5. Post-processing: Softmax activation, label assignment, confidence calculation
  6. Report Generation: Text report based on verdict and confidence
  7. Response: JSON with label, confidence, face_crop (base64), and report

Training Pipeline

  • Dataset: FaceForensics++ with 20,000+ images (80% train, 13% val, 7% test)
  • Augmentation: Horizontal flip, rotation ±15°, color jitter, Gaussian noise, compression
  • Optimizer: Adam with learning rate 0.001
  • Loss: CrossEntropyLoss
  • Scheduler: ReduceLROnPlateau (factor=0.5, patience=3)
  • Epochs: 20 (with early stopping based on validation accuracy)
  • Batch Size: 64

Description: MesoNet CNN architecture for deepfake detection

import torch import torch.nn as nn class MesoNet(nn.Module): """ MesoNet: Lightweight CNN for deepfake detection Paper: "MesoNet: a Compact Facial Video Forgery Detection Network" (2018) """ def __init__(self, num_classes=2): super(MesoNet, self).__init__() # Feature extractor (Convolutional layers) self.features = nn.Sequential( # Block 1: Input 150x150x3 → Output 75x75x8 nn.Conv2d(3, 8, kernel_size=3, padding=1), nn.ReLU(inplace=True), nn.BatchNorm2d(8), nn.MaxPool2d(kernel_size=2, stride=2), # Block 2: Input 75x75x8 → Output 37x37x8 nn.Conv2d(8, 8, kernel_size=5, padding=2), nn.ReLU(inplace=True), nn.BatchNorm2d(8), nn.MaxPool2d(kernel_size=2, stride=2), # Block 3: Input 37x37x8 → Output 18x18x16 nn.Conv2d(8, 16, kernel_size=5, padding=2), nn.ReLU(inplace=True), nn.BatchNorm2d(16), nn.MaxPool2d(kernel_size=2, stride=2), # Block 4: Input 18x18x16 → Output 9x9x16 nn.Conv2d(16, 16, kernel_size=5, padding=2), nn.ReLU(inplace=True), nn.BatchNorm2d(16), nn.MaxPool2d(kernel_size=2, stride=2), ) # Classifier (Fully connected layers) self.classifier = nn.Sequential( nn.Flatten(), nn.Linear(16 * 9 * 9, 16), nn.ReLU(inplace=True), nn.Dropout(0.5), nn.Linear(16, num_classes) ) def forward(self, x): x = self.features(x) x = self.classifier(x) return x if __name__ == "__main__": model = MesoNet(num_classes=2) total_params = sum(p.numel() for p in model.parameters()) trainable_params = sum(p.numel() for p in model.parameters() if p.requires_grad) print(f"Total parameters: {total_params:,}") print(f"Trainable parameters: {trainable_params:,}") dummy_input = torch.randn(1, 3, 150, 150) output = model(dummy_input) print(f"Output shape: {output.shape}") print("✓ Model architecture verified!")
face_utils.py

Description: Haar Cascade face detection and cropping utilities

import cv2 import numpy as np face_cascade = cv2.CascadeClassifier(cv2.data.haarcascades + "haarcascade_frontalface_default.xml") def extract_face(image_bgr, target_size=(150, 150), padding=0.2): """ Takes a BGR numpy array, returns a cropped + resized face (BGR). Returns None if no face found. """ gray = cv2.cvtColor(image_bgr, cv2.COLOR_BGR2GRAY) h, w = image_bgr.shape[:2] faces = face_cascade.detectMultiScale( gray, scaleFactor=1.1, minNeighbors=5, minSize=(60, 60) ) if len(faces) == 0: return None # Select largest face faces = sorted(faces, key=lambda f: f[2] * f[3], reverse=True) x, y, fw, fh = faces[0] # Add padding pad_x = int(fw * padding) pad_y = int(fh * padding) x1 = max(0, x - pad_x) y1 = max(0, y - pad_y) x2 = min(w, x + fw + pad_x) y2 = min(h, y + fh + pad_y) face_crop = image_bgr[y1:y2, x1:x2] face_resized = cv2.resize(face_crop, target_size) return face_resized

" f"Note: no detector is 100% accurate — sophisticated deepfakes may still evade detection." ) elif media_type == "video": frames = result["frames_analyzed"] fake_ratio = result["fake_frame_ratio"] * 100 duration = result.get("duration_seconds", "unknown") if label == "FAKE": return ( f"VERDICT: MANIPULATED CONTENT DETECTED\n\n" f"The model analyzed {frames} sampled frames from a {duration}s video. 🔬 Technical Details

How MesoNet Detects Deepfakes

MesoNet operates on the principle that deepfake generation methods leave mesoscopic-level artifacts — patterns visible at medium scales (not pixel-level nor image-level, but in between).

Key Detection Signals:

  1. Texture Inconsistencies: AI-generated skin is often too smooth or has unnatural pore patterns
  2. Boundary Artifacts: Face-background transitions show blurring or color bleeding
  3. Lighting Mismatches: Different parts of the face may have inconsistent light direction
  4. Asymmetry Anomalies: Perfect symmetry (too perfect) or unusual asymmetry patterns
  5. Eye Inconsistencies: Reflections, iris details, or sclera color mismatches between eyes
  6. Compression Artifacts: Different compression levels in manipulated vs. real regions

Why 150×150 Resolution?

  • Preserves enough detail for artifact detection
  • Small enough for fast inference (~10ms on GPU)
  • Matches FaceForensics++ preprocessing
  • Reduces overfitting compared to higher resolutions

Data Augmentation Rationale

The augmentation pipeline simulates real-world variations:

  • Geometric: Handles face alignment variations
  • Color: Adapts to different lighting/cameras
  • Noise: Robust to compression and sensor noise
  • Blur: Works with out-of-focus or motion-blurred images
  • Compression: Handles social media/video compression

Limitations & Future Work

Known Limitations:
  • Trained primarily on Western faces (may have demographic bias)
  • May struggle with 2024+ generation methods (Stable Diffusion, etc.)
  • Heavy JPEG compression can cause false positives
  • Only analyzes largest face in multi-face images
  • Requires frontal or near-frontal face views
Potential Improvements:
  • Add Grad-CAM heatmaps for explainability
  • Implement temporal analysis for video (not just frame averaging)
  • Multi-face detection and individual verdicts
  • Ensemble with other architectures (EfficientNet, ViT)
  • Train on more diverse datasets (different demographics, generation methods)
